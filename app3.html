Features
  • Fully customizable templates: with each template, you can modify to create the most beautiful collage that is suitable for your photos.
  • Import photos from gallery or take from camera. You can choose existing photos in gallery by using the default gallery, or using any file manager app like ES File Manager, IO File Manager, Root Browser,...
  • Apply Instagram liked effects to your photo: there are many effects for you to apply to your photo.
  • Decorate the collage: you can add text, balloon text to the photo.
  • A built-in gallery for browsing created collage photos.
  • rowse the collage templates with different options: browse default templates, browse favorite templates, browse recently used templates or browse customized templates.
  • Share created collages with friends via Facebook, SMS, Mail,...
